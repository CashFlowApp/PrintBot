Polymarket YES/NO Arbitrage Bot — Windows (PowerShell)

1. Open PowerShell in the project folder:
   cd "c:\Users\micha\OneDrive\Desktop\FinalArbBot"

2. Create virtual environment (recommended):
   python -m venv .venv
   .\.venv\Scripts\Activate.ps1

3. Install dependencies:
   pip install -r requirements.txt

4. Copy environment template and edit (never commit .env):
   copy .env.example .env
   # Edit .env with your API keys and parameters.

5. Validate only (Gamma + fees + both WebSockets, then exit — no trading):
   python -m bot.validate_only

6. Run the bot (default: paper mode; no real orders):
   python run.py

7. To stop cleanly: create a file named "KILL" in the project directory,
   or press Ctrl+C. The bot checks for KILL on each cycle.

All secrets must come from .env only. PAPER_MODE=true by default.
